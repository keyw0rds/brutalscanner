### How To Use?
=
> Install Packages : Python v3 
> Save File In Directory C:\
> If Packages Was Installed, RUN using command. 
$ py brutalscanner.py
----
Our Command :
scan : To start scanning file.
remove : To remove file.
exit : close tools.

Example :
cmd> scan
cmd> exit
cmd> remove

file> filename.txt
or
file> directory/filename.txt
